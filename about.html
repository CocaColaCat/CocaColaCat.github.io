说说软件开发

  • 敏捷 - 剪掉不重要的功能，快速迭代
  • REST API 开发 - 写测试，写测试，写测试，重要的事情说三次
  • 微信开发 - 搭建本地测试，远离 staging 或者 production
  • 上线部署 - 自动化和下班前不要部署（因为说好5分钟的部署，但是却会花一个小时）
  • 监控 - 没有监控就没有提高，用数据来指导改善
